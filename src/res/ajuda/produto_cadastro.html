Cadastro de Produto

Principal

  • Id: Número sequencial gerado automaticamente pelo sistema.
  • Código: Informe um código próprio, código de barras da indústria ou deixe em branco. Quando deixado em branco sistema preencherá com o id.
  • Localização: Utilize para indicar o local onde o produto é armazenado. Ex: Galpão B, Prateleira 4
  • Conteúdo: Deve ser preenchido se este item for componente de outro produto. Será usado no cálculo do estoque. Exemplo: produto Saco de 50kg (cadastrar o conteúdo = 50kg) -> mesmo produto por kg (cadastrar o Saco de 50kg como componente deste)
  • Item de balança: Define se o item será incluído no arquivo de exportação para balança. As unidades de medida diferentes de KG são exportadas como Unitário.
  • Dias Validade: Utilizado no arquivo de balança.
  • Aplicação: Utilize para descrever a utilização do produto.
  • Produto pode ser montado: Produtos como pizzas que podem ser montados junto com outros no momento da venda.
  • Tamanhos: Use para informar valores diferentes de acordo com o tamanho. Os tamanhos são cadastrados dentro das categorias.

Dados Fiscais SAT

As informações fiscais devem sempre ser preenchidas sob a orientação de um contador
  • Origem: Origem do produto.
  • NCM: Nomenclatura Comum do Mercosul. Utilize a lupa se necessário.
  • CEST Código Especificador da Substituição Tributária. Utilize a lupa se necessário. Preenchimento não obrigatório.
  • CFOP: Código Fiscal de Operações e Prestações.
  • Situação Tributária ICMS: As opções são exibidas de acordo com o regime Tributário configurado em Sistema.

Dados Fiscais NF-e

IPI

  • CNPJ do Produtor CNPJ do produtor da mercadoria, quando diferente do emitente. Somente para os casos de exportação direta ou indireta.
